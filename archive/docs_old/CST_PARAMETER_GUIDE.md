# CST 參數調整指南

本文件說明 CST (Class Shape Transformation) 方法的參數及其對幾何形狀的影響。

---

## 📐 CST 數學原理

CST 方法將機身半徑分佈定義為：

$$R(\psi) = C(\psi) \cdot S(\psi) \cdot L$$

其中：
- $\psi = x/L$ : 無因次縱向位置 (0 到 1)
- $L$ : 機身長度
- $C(\psi)$ : **Class Function**（控制頭尾形狀）
- $S(\psi)$ : **Shape Function**（控制局部變化）

---

## 1. Class Function 參數

### **N1（機頭參數）**

控制機頭的尖銳程度。

$$C(\psi) = \psi^{N1} \cdot (1-\psi)^{N2}$$

| N1 值 | 效果 | 適用場景 | 測試案例 |
|-------|------|----------|----------|
| 0.3 | 非常尖銳的鼻錐 | 高速設計（但本專案不適用）| Test_3_SharpNose |
| **0.5** | **橢圓形鼻錐（標準）** | **推薦值，氣流平順** | Test_1_Baseline |
| 0.7 | 鈍圓的鼻錐 | 增加內部空間，但阻力較高 | Test_4_BluntNose |
| 1.0 | 完全圓頭 | 不推薦（停滯點壓力損失大）| - |

**視覺效果**：
- N1 ↓ → 鼻錐更尖 → 座艙空間更前移
- N1 ↑ → 鼻錐更鈍 → 前段較粗短

**實測數據（2.5m 機身）**：
- N1=0.3: 肩膀寬度 0.352m（偏窄）
- N1=0.5: 肩膀寬度 0.341m
- N1=0.7: 肩膀寬度 0.384m

---

### **N2（機尾參數）**

控制機尾的收縮速度。

| N2 值 | 效果 | 適用場景 | 測試案例 |
|-------|------|----------|----------|
| 0.8 | 較鈍的尾錐 | 增加尾段內部空間 | Test_4_BluntNose |
| **1.0** | **尖銳尾錐（標準）** | **滿足 Kutta 條件，推薦** | Test_1_Baseline |
| 1.2 | 非常尖銳 | 易產生氣流分離（不推薦）| - |

**視覺效果**：
- N2 ↑ → 尾錐更尖 → 尾流區更小（理想上）
- N2 ↓ → 尾錐較鈍 → 可能產生亂流

**推薦組合**：
```python
"n_nose": 0.5,  # 橢圓鼻錐
"n_tail": 1.0,  # 尖銳尾錐
```

## 2. Shape Function 參數（Bernstein 權重）

### **權重向量（Weights）**

使用 4 階 Bernstein 多項式：

$$S(\psi) = \sum_{i=0}^{3} w_i \cdot B_i^3(\psi)$$

其中 $B_i^3(\psi)$ 是 Bernstein 基函數。

### **權重值範圍與效果**

| 權重組合 | 效果 | 肩膀寬度 | 測試案例 |
|----------|------|----------|----------|
| [0.15, 0.20, 0.20, 0.05] | 標準設計 | 0.341m | Test_1_Baseline |
| [0.10, 0.18, 0.18, 0.05] | 較細長 | 0.352m | Test_3_SharpNose |
| [0.20, 0.25, 0.22, 0.08] | 較粗壯 | 0.384m | Test_4_BluntNose |
| **[0.25, 0.35, 0.30, 0.10]** | **大空間（推薦）** | **0.565m** | Test_5_HighWeights |

### **各權重的物理意義**

```python
width_weights = [w0, w1, w2, w3]
```

- **w0**: 控制前段（0-25%）的粗細
- **w1**: 控制座艙區（25-50%）的隆起程度 ⭐
- **w2**: 控制後段（50-75%）的收縮速度
- **w3**: 控制尾段（75-100%）的形狀

**調整策略**：
1. **增加肩膀空間** → 提高 w1 和 w2
2. **減少阻力** → 降低所有權重（但要確保空間足夠）
3. **平衡設計** → w1 > w2 > w0 > w3

---

## 3. 超橢圓參數

### **Super_M 和 Super_N**

控制橫截面的形狀。超橢圓方程：

$$\left| \frac{y}{a} \right|^n + \left| \frac{z}{b} \right|^n = 1$$

| 參數值 | 形狀 | 空間利用率 | 適用場景 |
|--------|------|------------|----------|
| 2.0 | 標準橢圓 | 中等 | 傳統設計 |
| **2.5** | **略方形（推薦）** | **良好** | **平衡空間與阻力** |
| 2.8 | 明顯方形 | 最大 | 需要極大空間時 |
| 3.0 | 接近矩形 | 極大 | 可能增加阻力 |

**重要發現**：
⚠️ **超橢圓指數不影響 CST 計算的半徑值！**
- Test_1 (super_n=2.0) 和 Test_2 (super_n=2.8) 的肩膀寬度相同（0.341m）
- 超橢圓只改變截面的**角落形狀**，不改變**最大寬度**

**視覺效果**：
```
n=2.0:  ╱─╲     標準橢圓
n=2.5:  ┌─┐     方圓形
n=3.0:  ┏━┓     接近矩形
```

## 4. 測試結果總結

### **肩膀寬度需求：≥ 0.45m**

| 設計 | N1 | N2 | 權重組合 | 肩膀寬度 | 是否合格 |
|------|----|----|----------|----------|----------|
| Test_1 | 0.5 | 1.0 | [0.15, 0.20, 0.20, 0.05] | 0.341m | ❌ |
| Test_2 | 0.5 | 1.0 | [0.15, 0.20, 0.20, 0.05] | 0.341m | ❌ |
| Test_3 | 0.3 | 1.0 | [0.10, 0.18, 0.18, 0.05] | 0.352m | ❌ |
| Test_4 | 0.7 | 0.8 | [0.20, 0.25, 0.22, 0.08] | 0.384m | ❌ |
| **Test_5** | **0.5** | **1.0** | **[0.25, 0.35, 0.30, 0.10]** | **0.565m** | **✅** |

### **關鍵發現**

1. **權重值是決定性因素**
   - 目前測試的標準權重（0.15-0.20）太小
   - 需要提高到 0.25-0.35 才能滿足肩膀寬度要求

2. **N1 影響有限**
   - N1 從 0.3 到 0.7，肩膀寬度僅變化 0.03m
   - 權重的影響遠大於 N1/N2

3. **超橢圓的真實作用**
   - 不改變最大寬度
   - 只優化角落空間（肩膀、手肘區域）
   - 建議使用 2.5-2.8

```python

gene = [

```
